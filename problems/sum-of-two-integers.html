Sum of Two Integers

Statement

Given two integers a and b, return the sum of the two integers without using the operators + and -.

 

Example 1:

Input: a = 1, b = 2
Output: 3

Example 2:

Input: a = 2, b = 3
Output: 5

 

Constraints:

  • -1000 <= a, b <= 1000
Tags: Math, Bit Manipulation

Solution

The key idea is to use bitwise operators to simulate addition. The & operator can be used to find the carry bits, and the ^ operator can be used to find the sum bits (without carry). We then shift the carry bits one position to the left (<< 1) and repeat the process until there are no more carry bits. 

public class Solution {
    public int getSum(int a, int b) {
        while (b != 0) {
            int carry = a & b;
            a = a ^ b;
            b = carry << 1;
        }
        return a;
    }
}
